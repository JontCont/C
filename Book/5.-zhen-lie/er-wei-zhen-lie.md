# 二維陣列

列為橫向，行為直向。類似於隊伍中或是教室桌椅的形式。

二維陣列宣告方式 :

{% hint style="info" %}
資料型態　陣列名稱　\[ 列數 m \] \[ 行數 n \]
{% endhint %}

### 說明

1. 資料型態
2. 陣列名稱
3. 列數m
4. 行數n 
5. 二維陣列有兩個\[ \]
6. 一維度索引範圍於0 - m-1 之間
7. 二維度索引範圍於0 - n-1 之間

### 初始化

{% hint style="info" %}
資料型態　陣列名稱［列數 m］［行數 n］ =  
{ 初始值 a 11, 初始值 a12 ,初始值a13...},

{ 初始值 a 21, 初始值 a22 ,初始值a23... }

....

{ 初始值 a 1, 初始值 amn... ,初始值amn };
{% endhint %}
